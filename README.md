# 📊 Sales Dashboard Project (Excel)

## 🔹 Overview

This project focuses on analyzing sales data using Excel. The goal was to transform raw data into meaningful insights using Pivot Tables, Charts, and an Interactive Dashboard.

## 🔹 Tools Used

* Microsoft Excel
* Pivot Tables
* Charts (Bar, Pie, Line)
* Slicers (Interactive Filters)

## 🔹 Key Features

* KPI Cards (Total Sales, Total Orders, Average Order Value)
* Sales analysis by City and Category
* Monthly Sales Trend
* Interactive dashboard using slicers

## 🔹 Insights

* Electronics category contributes the highest revenue
* Pune generates the highest sales among all cities
* Sales show variation across months indicating seasonal trends
* High-value products drive overall revenue growth

## 🔹 Conclusion

The dashboard helps in quick decision-making by providing clear visual insights and interactive filtering options.
